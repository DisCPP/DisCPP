Constructs a discpp::Guild object from an id.

This constructor searches the guild cache to get a guild object.

If you set can_request to true, and the message is not found in cache, then we will request the message from the REST API. But if its not true, and its not found, an exception will be thrown of DiscordObjectNotFound.

Parameters
[in]idThe id of the guild.
[in]can_requestWhether or not the library can request the message from the REST API.
Returns
